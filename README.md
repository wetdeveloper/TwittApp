# 🚀 TwittApp — Modern Social Network Platform

> ⚠️ **Project Status: Beta (Actively Under Development)**

TwittApp is a modern social networking platform inspired by Twitter/X, designed with a focus on speed, simplicity, and real-time user interactions.

Users can create posts, interact with others, follow accounts, send direct messages, and manage their profiles through a clean and responsive interface.

---

## ✨ Features

### 📝 Posts & Interactions
- ⚡ Real-time tweeting with AJAX
- ❤️ Like system with heart animation
- 🔁 Retweet functionality
- 💬 Multi-level comments and replies
- 📱 Responsive timeline experience

### 👤 User System
- User profile management
- 🖼️ Profile picture upload
- Follow / Unfollow system
- User activity system

### 💬 Messaging
- Direct messaging between users
- Fast and simple conversations

### 🎨 Interface
- Modern and clean UI
- Built with Tailwind CSS
- Smooth animations
- Fully responsive design

# ⚡ Quick Start

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/TwittApp.git

cd TwittApp
```

Create a virtual environment:

### Linux / macOS

```bash
source venv/bin/activate
```

### Windows

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python3 run.py
```
