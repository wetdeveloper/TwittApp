
# 🕊️ **TwittApp** - Modern Twitter Clone

![Banner](https://via.placeholder.com/1200x400/1da1f2/ffffff?text=TwittApp+-+Modern+Twitter+Clone)

A beautiful, fast, and fully functional Twitter clone built with **Flask**, **Tailwind CSS**, and modern AJAX.

> **⚠️ Currently in active development (Beta Stage)**

---

## ✨ Features

- Real-time tweeting with AJAX
- Like system with heart animation
- Multi-level comments & replies
- Direct messaging
- Follow / Unfollow system
- Profile picture upload
- Clean, modern & responsive UI
